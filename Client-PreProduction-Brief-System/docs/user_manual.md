# DigiQuest pre-production portal - User Manual

This manual details features and workflows for each account role inside the Pre-Production Brief Portal.

---

## 1. Client Partner Guide

### Creating and Submitting Briefs
1. **Onboard**: Register via the landing page or receive login credentials.
2. **Draft a brief**: Select **Submit Brief** from the sidebar. Fill in Basic details, Timeline boundaries, Production requirements, and Approval contacts.
3. **Upload files**: In step 4, upload script text, brand guidelines, and reference images.
4. **Save Draft**: Click **Save as Draft** if requirements are still shifting. You can modify it later.
5. **Form Submission**: Click **Submit Pre-Prod Brief**. Note: all fields must be complete before formal submission.

### Revision Workflows
1. If a Project Manager requests revisions, you will receive an in-app alert and email.
2. Your brief status changes to **Revision Requested**.
3. Open the brief detail tab, read comments, make modifications by clicking **Modify Brief**, and resubmit it.

---

## 2. Project Manager & Admin Guide

### Reviewing Briefs
1. Go to **Project Briefs** to view incoming submissions. Filter by status **Submitted**.
2. Click the project title to inspect description details, timelines, scripts, and download attachments.
3. If specs look correct, click **Move to Under Review**.

### Collaboration & Approval
1. Add comments in the right sidebar. You can type `@username` to mention other crew members or the client.
2. If specs need adjustments, click **Move to Revision Requested** and add a comment indicating missing criteria.
3. Once specifications align with studio resources, click **Move to Approved**.
4. Once production completes or is scheduled, click **Move to Archived**.

### Reports & Audits (Admin Only)
* Open **Reports & Analytics** to track average approval times and submission counts.
* Open **System Audit Logs** to audit data entries, credentials updates, or delete histories.
